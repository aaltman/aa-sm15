: 3.2.8.7.1.1. Node Types

A graph node can be one of:

  • kernel

  • CPU function call

  • memory copy

  • memset

  • empty node

  • waiting on an event

  • recording an event

  • signalling an external semaphore

  • waiting on an external semaphore

  • conditional node
