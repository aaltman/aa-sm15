Valid Parenthesis String - Leetcode 678 - Python


Basic cases are:

 - if a left paren, track it

 - if an asterisk, track it

 - if a right paren, then either:

   - match it with a left paren

   - match it with an asterisk

   - or, if those can't be done, then end the algorithm.

 

Once we've matched all cases in the input string, then we can check tracked lefts and asterisks.  If there's an asterisk that came before a left paren after tracking and eliminating previous matches, then fail.  Otherwise, if we've matched everything that we've tracked, then return true.
