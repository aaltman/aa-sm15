: Example analysis of an M/M/1 queue[edit]

Consider a queue with one server and the following characteristics:

  • λ: the arrival rate (the reciprocal of the expected time between each customer arriving, e.g. 10 customers per second)
  • μ: the reciprocal of the mean service time (the expected number of consecutive service completions per the same unit time, e.g. per 30 seconds)
  • n: the parameter characterizing the number of customers in the system
  • Pn: the probability of there being n customers in the system in steady state

Further, let En represent the number of times the system enters state n, and Ln represent the number of times the system leaves state n. Then |En−Ln|∈{0,1} for all n. That is, the number of times the system leaves a state differs by at most 1 from the number of times it enters that state, since it will either return into that state at some time in the future (En=Ln) or not (|En−Ln|=1).

When the system arrives at a steady state, the arrival rate should be equal to the departure rate.

Thus the balance equations

μP1=λP0
λP0+μP2=(λ+μ)P1
λPn−1+μPn+1=(λ+μ)Pn

imply

Pn=λμPn−1, n=1,2,…

The fact that P0+P1+⋯=1 leads to the geometric distribution formula

Pn=(1−ρ)ρn

where ρ=λμ<1.
